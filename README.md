# 🏥 Hospital Emergency Room Dashboard

## Overview
The Hospital Emergency Room Dashboard is an Excel-based interactive dashboard designed to analyze and monitor emergency room operations. It provides insights into patient flow, admissions, wait times, patient demographics, and department referrals through dynamic visualizations and filters.

## Key Performance Indicators (KPIs)
- Number of Patients
- Average Wait Time
- Patient Rating

## Dashboard Features
- Patient Attendance Analysis
- Admission Status Analysis
- Gender-wise Patient Distribution
- Age Group Analysis
- Department Referral Analysis
- Year-wise Filtering
- Month-wise Filtering

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
